# CodeAlpha Artificial Intelligence Internship Projects

This repository contains the projects completed during the CodeAlpha Artificial Intelligence Internship.

---

# Projects Included

## 1. Object Detection and Tracking

A real-time AI-based object detection and tracking system using YOLOv8 and SORT tracking algorithm.

### Features
- Real-time object detection
- Object tracking with unique IDs
- Video/webcam support
- Bounding boxes and labels
- AI-powered computer vision system

### Technologies Used
- Python
- OpenCV
- YOLOv8
- SORT Tracking
- NumPy

---

## 2. AI FAQ Chatbot

An NLP-based FAQ chatbot that answers user questions using cosine similarity and machine learning techniques.

### Features
- AI chatbot interface
- FAQ matching
- NLP preprocessing
- Cosine similarity matching
- Modern dark UI

### Technologies Used
- Python
- NLTK
- Scikit-learn
- CustomTkinter

---

## 3. AI Language Translator

A smart language translation tool with multilingual support using Google Translation API.

### Features
- Translate text between languages
- Source and target language selection
- Copy translated text
- Modern dark UI
- Swap language functionality

### Technologies Used
- Python
- Deep Translator
- CustomTkinter
- Pyperclip
